# Security Policy

## Supported Versions

Only the latest version of **SecureCode Reasoning Agent** is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 2.0.x   | Yes       |
| < 2.0.0 | No        |

## Reporting a Vulnerability

We take the security of this project seriously. Because this is a hackathon MVP project, please report security issues through GitHub issues or by directly contacting the repository maintainers.

Please do **NOT** submit real production credentials, company data, or proprietary secrets when filing security reports.
